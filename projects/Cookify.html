Cookify

  • Tech Stack: HTML, CSS, Javascript, MongoDB, ExpressJS, Node.js & React.js
  • Github URL: Project Link

This web application is a recipe management system that allows users to create, store and manage their favorite recipes in one convenient location. The inspiration for this project came from a passion for cooking and the desire to record and share delicious recipes with others. Built using modern web development technologies, including the MERN stack (MongoDB, ExpressJS, React v6.0, & Node.js), this CRUD application utilizes RESTful architecture principles to provide seamless user experience. Upon logging in, users are presented with an index view of all the recipes in the foods list. They can browse through their collection of recipes or add a new recipe by clicking the "Create" button on the navigation bar. This takes them to a new page where they can enter details about the recipe such as the name, cuisine type, ingredients, and cook time. Once submitted, the recipe is saved in the index view for future reference. Users can view the details of each recipe by clicking on the recipe name, and from there, they can update or delete the recipe as needed. If a user decides to update a recipe, they are redirected to a pre-populated form that they can modify to reflect the changes they want to make. Once the changes are made, they can click the "Update" button and be redirected back to the recipe details page with the updated information. If a user decides to delete a recipe, they can simply click the "Delete" button on the details page and the recipe will be removed from the index view. Overall, this recipe management system is a user-friendly and efficient way to organize and share delicious recipes with others.
