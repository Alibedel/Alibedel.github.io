Cookify

  • Type: Recipe management app
  • Tech Stack: MongoDB, Express, React, Node.js (MERN), HTML, CSS
  • Code: github.com/Alibedel/coofiy
Overview

Cookify is a recipe management system that lets users create, store, and manage their favorite recipes in one convenient place. The project grew out of a passion for cooking and the desire to record and share great recipes with others.

Key Features

After logging in, users see an index of all recipes and can add new ones — name, cuisine type, ingredients, and cook time. Clicking a recipe opens its details, where users can update it through a pre-populated form or delete it entirely. Every action flows through a clean, minimal-click interface.

What This Project Shows

A complete CRUD application on the MERN stack (MongoDB, Express, React, Node.js) following RESTful architecture principles — component-based UI, API design, and the end-to-end product workflow I bring to client web apps.
