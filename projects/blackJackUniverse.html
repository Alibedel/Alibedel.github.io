BlackJack Universe

  • Type: Browser game
  • Tech Stack: HTML, CSS, Bootstrap, JavaScript, GitHub Pages
  • Code: github.com/Alibedel/Blackjack-Universe
Overview

BlackJack Universe is a fully playable blackjack game that runs entirely in the browser. The objective is classic: get as close to 21 as possible without busting, playing against the dealer — winner takes all.

How It Works

Players start with two cards and choose to "hit" or "stand" to get closer to 21. The dealer draws in turn, and the round resolves when a player reaches 21 or busts. The game includes a scoring system to track progress across rounds and a reset option to start fresh.

What This Project Shows

Building a complete game loop in vanilla JavaScript — state management, DOM manipulation, and translating a real-world rule set into reliable program logic — without leaning on frameworks. Skills that carry directly into interactive features for client websites.
