Hoops Nation

  • Type: Full-stack web app (NBA league tracker)
  • Tech Stack: HTML, CSS, JavaScript, MongoDB, Express, Node.js
  • Code: github.com/Alibedel/Hoops-Nation
  • Live Demo: hoops-nation.onrender.com
Overview

Hoops Nation is an NBA league tracker that lets basketball fans stay up to date with the latest standings of their favorite teams. Users can track standings, view and add players, and connect with other fans who support the same team.

Key Features

The app tracks NBA league standings, supports viewing and adding players to the system, and lets fans read and post comments on their team's performance. It's a complete CRUD application built with RESTful architecture principles, designed to be robust and scalable with large amounts of data.

What This Project Shows

Full-stack development on the MEN stack (MongoDB, Express, Node.js): designing data models, building REST routes, and shipping a complete product — the same workflow I use for client dashboards and database-driven business sites.
