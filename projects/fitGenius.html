Fitness Genius

  • Type: Workout & diet tracking app
  • Tech Stack: Python, Django, PostgreSQL, AWS (photo uploads)
  • Code: github.com/Alibedel/Fitness-Genius
Overview

Fitness Genius is a workout tracker that helps users create, track, and maintain their workout routines, diet, and progress — replacing scattered notes with one clean tool. Users can create, edit, and delete workouts, add routines and diets, and upload progress photos.

Key Features

After signing up and logging in, users see an index of all their workouts and can drill into any workout's details. From there they can add routines (name, sets, reps) and diets (food, grams), and edit or delete anything. Progress photos are uploaded and stored via AWS, giving users a visual record of their fitness journey.

What This Project Shows

A full CRUD application built with Python, Django, and PostgreSQL under RESTful architecture principles — user authentication, per-user data modeling, file storage in the cloud, and a UI organized around how people actually train.
