Iconic Tech Dashboard

  • Type: Client project — web-based database management system
  • Tech Stack: Django, Python, MySQL, JavaScript, HTML, CSS, Bootstrap
  • Code: github.com/Liban435/IconicTech
  • Client: Iconic Tech
Introduction

The store owners had been using outdated methods to manage their retail business and wanted to modernize — putting technology in place that could scale a small mom-and-pop shop toward a franchise. The branding was already franchise-ready; the systems behind the scenes were not. My job was to build the technical foundation to match.

Project Description

The Iconic Tech Dashboard is a CRUD application serving as the store's core database management software, with a user-friendly, visually clean GUI. It revolves around two main tasks: managing store data — inventory, invoices, customer and employee information — and running business queries, like which cashier sold the most in October or which products have the best profit margins. Queries are developer-programmed options, with the system designed so new ones can be added after launch.

The project follows a layered architecture with three layers: the database schema (MySQL), a data access layer (Python), and a presentation layer (Django web app).

Challenges

The store's existing data lived in spreadsheets and couldn't be processed directly — it needed a major cleaning operation. Rather than start there, the project focused on building a new system with a data-entry protocol (enforced in software and documented as a staff manual) to guarantee high-quality data going forward. Only once clean data was accumulating did work begin on the legacy datasets. Inventory also lacked unique IDs and pricing, so the workflow was streamlined for digitizing stock, paired with physical barcode labeling.

Milestones

Core relational database in MySQL with all entities and relations defined and tested; an initial set of SQL queries with room to grow post-launch; the Python data access layer; the Django web app; two client revision rounds; and final launch of the Iconic Tech App.
